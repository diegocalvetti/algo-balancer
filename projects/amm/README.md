# 🧮 Balancer-style AMM on Algorand

## 🛠️ Installation

```bash
# Prerequisites:
# - Node.js 22+
# - AlgoKit CLI

# Install dependencies
npm i
```
## ⚙️ Run

```bash
# Compile all the contracts
npm run build
```

```bash
# Clone the example env
cp .env.example .env
```

```bash
# Interactive shell
npm run execute
```
